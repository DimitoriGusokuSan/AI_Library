# check-wid

Скилл для Claude Code — прозрачная проверка выполненной работы.

## Что делает

Когда вызываешь `/check-wid`, Claude разбирает всё, что сделал в текущем разговоре, по структуре:

1. **Что понял** — переформулировка задачи
2. **На чём основывался** — конкретные файлы, данные, ответы API
3. **Что сделал** — пошагово, с результатами
4. **Как проверил** — QA по двум вопросам: "выполнено?" и "верно?"

## Зачем

Ловит ошибки в понимании задачи, в выборе подхода и в исполнении. Убеждает, что результат получен осознанно, а не случайно.

## Установка

Скопируй `SKILL.md` в `.claude/skills/check-wid/SKILL.md` своего проекта.

## Использование

```
/check-wid
/check-wid задача №3
```
